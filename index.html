pitchfork by rtownsend

Free tool for making files available on a website or intranet. Includes HTTP file-browser; basic search and access control and zip file creation. Runs on Linux/Unix.

Easily share files on the internet or your intranet with Pitchfork, a free tool that displays and permits the download of files not normally accessible by your webserver. It features file-name searching, inverted index searching (planned for next release) and zip/tar file creation.

Dependencies

Pitchfork depends on PHP5, zip or tar commands, access to a command-line and a webserver to pull it all together (such as apache).
